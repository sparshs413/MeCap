MeCap

Abstract and Motivation

The main objective of this project is to develop a cheap extension of VR head-sets that can provide real-time estimates of a wearer’s 3D body-pose, hand pose,facial expressions which can be used for VR gameplay.The project aims to create a low cost VR headset using hemispherical mirror-ball and rear-view camera of phone. The video of spherical mirror is taken usingthe VR headset and is fed into an image pre-processing python code in OpenCVwhich unwraps the spherical image and produces perspective projection of it.This is then given as an input to a machine-learning algorithm implemented viatechniques of transfer-learning and OpenPose which detects the wearer’s bodyand hand pose as well as facial expressions. We have simulated a virtual envi-ronment for the same in Blender and implemented these steps on it for testingand debugging purpose.
